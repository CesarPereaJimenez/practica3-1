Javascript

Diseño y desarrollo web.-Javascript es un lenguaje de programacion que se utliza con frecuencia en diseño WEB
para generar efectos mas comlejos que no se puede alcanzar usando HTML.Javascript se utiliza para una gran
variedad de usos, pero una de las aplicaciones mas comunes es la creacion de los botones de navegacion que
cambian de color o aspecto cuando el usuario mueve el cursor sobre el botón.

de una weeb y invluir los hipervinculos o enlaces a otra paginas.Este lenguaje lo invento en 1991 el Doctor
